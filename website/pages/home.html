MPOOL is a Bitzeny mining pool
  • Server: Japan
  • Pool fee is 1%
  • Payment method is prop
  • Minimum payment is 1 ZNY
Getting Started
  • GPUで採掘する場合は こちら のマイニングソフトが良いと思います。
  • Download a miner. Gpu best miner is here
  • CPUで採掘する場合は こちらのマイニングソフトが良いと思います。
  • Cpu miner is here
  • ./ccminer -a yescryptr8 -o stratum+tcp://zny.mpool.jp:3333 -u [BITZENY ADDRESS] -p x
  • ./minerd -a yescrypt -o stratum+tcp://zny.mpool.jp:3333 -u [BITZENY ADDRESS] -p x
  • [BITZENY ADDRESS] is your bitzeny address
  • もしワーカー名を追加したい場合は、 ./ccminer -a yescryptr8 -o stratum+tcp://zny.mpool.jp:3333 -u [BITZENY ADDRESS].[WORKERNAME] -p x の様にアドレスの後に . ワーカー名を入れてください。
  • if workername add ./ccminer -a yescryptr8 -o stratum+tcp://zny.mpool.jp:3333 -u [BITZENY ADDRESS].[WORKERNAME] -p x
  • example : ./ccminer -a yescryptr8 -o stratum+tcp://zny.mpool.jp:3333 -u ZqvHJPYWe3Ed7CmVENj3rMxkqnd1VHePx2 -p x
